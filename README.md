Програмна система для інтелектуального моніторингу умов існування та стану здоров’я тварин на фермах FarmEnviroControl

Програмна система FarmEnviroControl призначена для інтелектуального моніторингу стану здоров’я та умов існування тварин на фермах із використанням технологій Інтернету речей (IoT). Система забезпечує автоматизований збір, передачу, збереження та аналіз біометричних показників тварин, таких як температура тіла, пульс та загальний фізіологічний стан, з подальшим формуванням стану тварини та сповіщень про критичні зміни. Основна мета розробки — підвищення ефективності контролю за станом тварин, зменшення людського фактору та забезпечення своєчасного реагування на потенційно небезпечні ситуації на фермах.

Система працює за принципом клієнт-серверної архітектури та включає IoT-клієнт, серверну частину та користувацький веб-інтерфейс. IoT-клієнт, реалізований на базі мікроконтролера ESP32, імітує або зчитує показники фізіологічного стану тварини та передає їх на сервер через мережеве з’єднання. У середовищі Wokwi реалізовано програмну логіку генерації температури та пульсу, визначення стану тварини (GOOD, NORMAL або BAD), а також локальну індикацію стану за допомогою світлодіода та звукового сигналу. У разі перевищення критичних порогів температури або пульсу система автоматично переводить тварину у критичний стан та активує сигнал тривоги.

Серверна частина FarmEnviroControl реалізована з використанням Node.js та фреймворку Express.js і виконує обробку вхідних даних, їх збереження у базі даних MongoDB та реалізацію бізнес-логіки системи. Сервер приймає структуровані JSON-повідомлення від IoT-клієнтів, аналізує значення показників, зберігає історію вимірювань та оновлює стан тварин. У разі фіксації критичних параметрів сервер формує відповідні сповіщення, які можуть бути використані для подальшого інформування користувачів або розширення системи аналітики.

Система підтримує роботу з фермами та тваринами як основними сутностями. Кожна ферма містить перелік тварин, а при реєстрації нової тварини вона автоматично прив’язується до вибраної ферми та додається до її структури у базі даних. Це дозволяє зберігати не лише окремі записи тварин, а й повну інформацію про склад ферми, кількість тварин та їх поточний стан. Такий підхід створює основу для подальшого розвитку функціональності, зокрема прив’язки ферм до користувацьких акаунтів і централізованого керування господарством.

Користувацький інтерфейс системи забезпечує взаємодію з серверною частиною через REST API та дозволяє переглядати інформацію про ферми, тварин і їхній поточний стан. У майбутньому інтерфейс може бути доповнений графіками історичних даних, розширеними сповіщеннями та інструментами аналітики для оцінки стану тварин у динаміці. Система спроєктована з урахуванням можливості масштабування, додавання нових типів сенсорів, розширення алгоритмів аналізу та інтеграції з іншими інформаційними платформами.

Таким чином, FarmEnviroControl є комплексним програмним рішенням для автоматизованого моніторингу стану тварин на фермах, яке поєднує IoT-технології, серверну обробку даних та основу для створення зручного користувацького інтерфейсу. Реалізована система демонструє практичну цінність для сучасного фермерського господарства, дозволяючи підвищити рівень контролю за тваринами, оперативно реагувати на зміни їхнього стану та закладає фундамент для розвитку інтелектуальних систем управління фермою.
