Програмна система для інтелектуального моніторингу стану здоров’я тварин VetMonitor

Опис теми: Система збирає та аналізує дані про стан здоров’я тварин (температура, пульс, активність та інші біометричні показники), що надходять з IoT-пристроїв у ветеринарних клініках. На основі цих даних система автоматично формує рекомендації для власників і ветеринарів, а також сповіщення про критичні зміни. Крім цього, платформа забезпечує зручне адміністрування користувачів, тварин і аналітичних алгоритмів.

Основна функціональність: Програмна система забезпечує збір біометричних даних у реальному часі, їх збереження, аналіз і формування рекомендацій. Також доступний інтерфейс для перегляду історії показників, списку сповіщень та стану кожної тварини. Передбачена система адміністрування для керування даними користувачів, тварин та ролей.

Складові проєкту:

Back-end Back-end відповідає за прийом та обробку даних, збереження інформації в базі даних і реалізацію бізнес-логіки. Функціонал включає: Прийом даних від IoT-пристроїв (температура, пульс, активність). Перевірку показників на вихід за порогові значення. Генерацію рекомендацій та автоматичних сповіщень. Маршрути для роботи з користувачами та тваринами (створення, редагування, видалення). Функції адміністрування та керування ролями користувачів. Захист даних та авторизацію адміністраторів та користувачів.

Front-end Front-end забезпечує інтерфейс для взаємодії користувача з системою. Основні можливості: Особистий кабінет для перегляду інформації про тварин. Панель стану здоров’я тварини з актуальними показниками та рекомендаціями. Список сповіщень щодо критичних змін. Перегляд історичних даних у вигляді графіків. Можливість для адміністратора керувати користувачами та тваринами.

IoT IoT-пристрої відповідають за збір та передачу біометричних даних у систему: Зчитування температури, пульсу, активності та інших сенсорних даних. Передача даних через MQTT у реальному часі. Можливість конфігурації інтервалу надсилання даних. Синхронізація із серверною частиною для формування рекомендацій. Оновлення прошивки з метою додавання нових сенсорів і функцій.
